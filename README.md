# 課題　 【　履歴登録アプリ　】

## ①課題内容（どんな作品か）
- いつ、どこで、どんな目的で会ったかを記録するアプリ

## ②工夫した点・こだわった点
- GeolocationとFirebaseを使用
- ボタンクリックで位置情報と現在時刻を取得
- 取得した情報をalertで表示

## ③難しかった点・次回トライしたいこと(又は機能)
- Realtime Databaseとの連携が不十分のままです。
- 本当はボタンクリックで日時やピンを保存して画面に表示したかったです。
